# Global Estate 🏡

Global Estate is a simple online real estate platform built using HTML, CSS, and JavaScript. This project provides worldwide property listings for **buying, selling, and renting** **villas** and **apartments** 🌍🏠. The platform includes **property categories** and a modern, user-friendly design.

## Features ✨
- **Property Categories**: Organized listings for easier browsing 📌.
- **Modern UI**: Clean and visually appealing interface 🎨.
- **Simple Navigation**: Easy-to-use layout for a seamless experience 🔍.
- **Desktop Design**: Optimized for desktop viewing 💻.

## Installation 🛠️
To view or contribute to this project, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/arshiya-mh/global-estate.git
   ```
2. Navigate into the project folder:
   ```sh
   cd global-estate
   ```
3. Open the `index.html` file in your browser to view the project.
